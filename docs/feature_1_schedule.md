# Feature 1: Schedule — UI экран расписания

> **Дата:** 2026-07-05  
> **Ветка:** `feature/1-schedule`

---

## Цель

Реализовать экран расписания мастер-классов, на котором клиент может просматривать доступные слоты на ближайшие 7 дней, видеть детальную информацию о каждом занятии и выбирать слот для записи.

## Требования (кратко)

| ID | Требование |
|----|------------|
| US-01 | Клиент видит расписание на ближайшие 7 дней |
| US-03 | Клиент видит детальную информацию о слоте (программа, мастер, свободные места, длительность) |
| EC-17 | Если расписания нет — показывается empty state «Пока нет доступных занятий» |
| EC-01 | При ошибке сети — показывается заглушка с кнопкой «Повторить» |
| R-027 | Дефолтный список слотов — 7 дней от текущего момента |

## Реализовано в коде

### 1. ScheduleViewModel (`lib/viewmodels/schedule_viewmodel.dart`)

- Наследует `ChangeNotifier` для реактивного обновления UI
- **Состояния:** `isLoading`, `error`, `isEmpty`, `slots`
- **Методы:**
  - `loadSchedule()` — загружает расписание из `MockApiService`, управляет флагами загрузки/ошибки
  - `retry()` — повторная загрузка после ошибки
- При загрузке симулируется сетевая задержка (600 мс)

### 2. ScheduleScreen (`lib/ui/screens/schedule_screen.dart`)

- StatefulWidget, подписанный на `ScheduleViewModel`
- **4 состояния отображения:**
  - **Loading** — скелетон-заглушка (`_ScheduleShimmer`) с 5 карточками-плейсхолдерами
  - **Error** — `EmptyState` с иконкой `wifi_off`, текстом ошибки и кнопкой «Повторить»
  - **Empty** — `EmptyState` с иконкой `event_busy` и текстом «Пока нет доступных занятий»
  - **Data** — `RefreshIndicator` (pull-to-refresh) + `ListView` с группировкой по датам
- **Группировка по датам:** слоты группируются по `DateTime` (день/месяц/год), каждая группа имеет заголовок с днём недели и датой (например, «Суббота, 5 июля» или «Сегодня, 5 июля»)
- Pull-to-refresh для обновления расписания

### 3. SlotCard (`lib/ui/widgets/slot_card.dart`)

- Карточка занятия с информацией:
  - **Бейдж с датой** (например, «5 июл») + время начала и окончания
  - **Название программы** (Ручная лепка / Гончарный круг)
  - **Мастер** — аватар (первая буква имени), имя, рейтинг (звёздочка + число)
  - **Длительность** (например, «2ч 30мин»)
  - **Свободные места** (например, «3 / 6») — красным цветом, если осталось ≤ 2 места
  - **Статус-чип:** «Отменено» (красный, для `cancelledByWorkshop`), «Мест нет» (серый, для `fullyBooked`)
- Неактивная (disabled) карточка для слотов, на которые нельзя записаться

### 4. EmptyState (`lib/ui/common/empty_state.dart`)

- Переиспользуемый виджет для пустых/ошибочных состояний
- Параметры: `icon`, `title`, `subtitle`, `actionLabel`, `onAction`
- Используется как для ошибки сети, так и для пустого расписания

### 5. main.dart

- Точка входа теперь показывает `ScheduleScreen` вместо заглушки
- Тема: Material 3, seed color `#8B5E3C` (терракотовый/глиняный)
