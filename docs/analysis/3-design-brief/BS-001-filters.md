## 1. Назначение и контекст

Шторка фильтров позволяет клиенту **уточнить список слотов** — не листать весь каталог
предстоящих мастер-классов, а быстро сузить его до релевантных вариантов (нужная дата,
программа, только со свободными местами, конкретный мастер).

Шторка открывается поверх [SCR-002 «Список слотов»](SCR-002-slot-list.md). Результат применяется
на **родительском экране**, не внутри шторки. BS-001 — **форма выбора параметров**, а не
отдельный экран результатов.

Контекст использования — дома и в мастерской ([foundations §1](00-foundations.md#1-продукт-и-аудитория)):
выбор должен быть крупным, быстрым и выполнимым одним пальцем.

---

## 2. Пользователь и задача дизайна

- **Роль:** Клиент.
- **Цель пользователя:** «Хочу видеть только подходящие мне занятия» — отфильтровать каталог,
  чтобы быстрее дойти до записи (US-3, P2).
- **Задача дизайна:** собрать комбинацию условий за минимум тапов; понятно показать выбранное;
  вернуть на список одним «Применить».

---

## 3. Навигация

**Вход:** тап **«Фильтры»** на [SCR-002](SCR-002-slot-list.md) → шторка снизу вверх.

**Выход:**

| Действие | Результат |
| :-- | :-- |
| **«Применить»** | Закрытие + обновлённый список на SCR-002 + индикатор активных фильтров |
| **«Сбросить»** | Сброс контролов к default внутри шторки |
| **Закрытие** (swipe / бэкдроп / ✕) | Закрытие **без** применения новых значений |

## 4. Контент в шторке

| Элемент | Что показываем | Default | Примечание |
| :-- | :-- | :-- | :-- |
| Заголовок | «Фильтры» | — | В хедере |
| **Дата старта** | Пресеты «Сегодня» / «Эта неделя» / «Выходные» + поля «С» / «По» | Без пресета (дефолт списка — 7 дней) | Прошедшие даты недоступны |
| **Программа** | Чипы: «Новичковая лепка», «Гончарный круг» и др. из справочника | Ничего не выбрано (любая) | Мультивыбор; ИЛИ внутри группы |
| **Только со свободными местами** | Toggle | **Выкл** | Скрывает слоты с «Мест нет» на SCR-002 |
| **Мастер** | Чипы имён мастеров из справочника | Ничего не выбрано (любой) | Мультивыбор; ИЛИ внутри группы |
| **«Применить»** | Primary CTA, фикс. внизу шторки | — | Всегда доступна |
| **«Сбросить»** | В хедере | Неактивна при default | — |

> **Комбинирование:** «И» между группами; «ИЛИ» внутри группы с мультивыбором (программа, мастер).

## 6. Компоненты и поведение

### 6.1 Выбор даты — пресеты + диапазон

- **Быстрые пресеты** (чипы): «Сегодня», «Эта неделя», «Выходные» — выбирается один; тап
  заполняет соответствующий диапазон.
- **Произвольный диапазон:** поля «С» и «По»; тап открывает date picker. Допускается одна дата (С = По).
- Выбор пресета и ручной диапазон синхронизированы (ручная правка снимает пресет).
- Прошедшие даты недоступны.

### 6.2 Программа — множественный выбор

- Чипы с мультивыбором из справочника (минимум: «Новичковая лепка», «Гончарный круг»).
- Состояния чипа: невыбран / выбран — не только цветом ([foundations §3.2](00-foundations.md#32-контраст)).

### 6.3 «Только со свободными местами»

- Toggle; по умолчанию **выключен**. Состояние считывается без опоры только на цвет.

### 6.4 Мастер — множественный выбор

- Чип-лист мастеров из справочника; без выбора — «любой».
- На время загрузки справочника — скелетон в группе (§7).

### 6.5 «Применить» / «Сбросить»

- «Применить» — закрывает шторку; родитель показывает индикатор активных фильтров / новую выдачу.
- «Сбросить» — только внутри шторки до «Применить»; активна, если хоть один параметр отличается от default.

> **Пустой результат** после «Применить» показывается на **[SCR-002](SCR-002-slot-list.md)**, не в шторке.

## 7. Состояния

| Состояние | Что рисуем |
| :-- | :-- |
| **Default** | Все контролы в значениях §4; «Сбросить» неактивна |
| **Есть выбор** | «Сбросить» активен; визуально видно выбранное |
| **Загрузка справочника мастеров** (если есть) | Скелетон в группе «Мастер»; остальное интерактивно |
| **Ошибка загрузки справочника** (опц.) | Неблокирующая ошибка с повтором; прочие фильтры доступны |

## 9. Доступность

- Грабер + swipe — не единственный способ закрытия; есть явная кнопка закрытия.
- CTA «Применить» достижим без скролла (фиксирован внизу шторки).
- Чипы и toggle — ≥ 44 pt, состояния не только цветом.
- Date picker удобен одной рукой; крупные области дней.

## 11. Зафиксированные UX-решения

| # | Решение |
| :-- | :-- |
| 1 | **Дата:** пресеты «Сегодня» / «Эта неделя» / «Выходные» + произвольный диапазон |
| 2 | **Программа и мастер — мультивыбор** (чипы); ИЛИ внутри группы, И между группами |
| 3 | **Живой счётчик результатов в шторке — не делаем.** Empty state — на SCR-002 после «Применить» |
| 4 | **Закрытие без «Применить»** не меняет ранее применённые фильтры на родителе |
| 5 | **«Сбросить» + «Применить»** возвращает дефолтную выдачу (7 дней, все программы и мастера) |
