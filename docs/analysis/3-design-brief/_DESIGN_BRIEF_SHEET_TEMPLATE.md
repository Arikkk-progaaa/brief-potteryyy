## 9. Доступность

- Грабер + swipe — не единственный способ закрытия; есть явная кнопка закрытия.
- CTA «Применить» достижим без скролла (фиксирован внизу шторки).
- Чипы и toggle — ≥ 44 pt, состояния не только цветом.
